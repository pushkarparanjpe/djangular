Usage:
------
git clone https://github.com/pushkarparanjpe/djangular  
cd djangular/  
python3 -m venv venv  
source venv/bin/activate  
pip install -r requirements.txt   
cd frontend/static/frontend/  
bower update  
cd ../../../../  
./manage.py migrate  
./manage.py runserver  

Visit: http://localhost:8000/
